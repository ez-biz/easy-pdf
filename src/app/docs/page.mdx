# EasyPDF Documentation

Welcome to the EasyPDF documentation! EasyPDF is a privacy-focused PDF manipulation suite built with Next.js 15.

## What is EasyPDF?

EasyPDF provides 17+ PDF tools that run entirely in your browser. Your files never leave your device, ensuring complete privacy.
